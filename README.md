# AI-Powered Customer Support & Service Automation Using Salesforce

A Salesforce-based CRM application that automates customer support and service management using AI-powered features. The project streamlines customer, merchandise, order, and case management through Salesforce automation.

## Features

- Customer & Order Management
- User Case Management
- Validation Rules
- Record-Triggered Flows
- AI Summary using Prompt Builder
- Reports & Dashboards

## Tech Stack

- Salesforce Lightning Platform
- Apex
- Flow Builder
- Prompt Builder
- Agentforce
